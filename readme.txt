== Description ==

Send SMS to Wordpress Users via Arsys that allows you to send SMS to your users.

If you have a Plugin that requires registration to your users, and you require a Mobile Phone, you can send them by groups a SMS.

Features include:

*   Send SMS to a group of registered users.
*   Use an Arsys Spanish Provider.
*   Send SMS Text Messages through WordPress dashboard.

We accept no responsibility for SMS messages send wrongfully, use at your own risk.

This is a WordPress plugin by [Closemarketing](http://www.closemarketing.es/ "Closemarketing") inspired by [David Perez](http://twitter.com/closemarketing "David Perez")

== Installation ==

= Installation =

1. Upload `send-sms-arsys` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

= Basic Setup =

1. Sign up with [Arsys WebSMS](http://bit.ly/WebSMSArsys "WebSMS Arys")
2. Enter your connection details under the SMS Manager -> Options menu in your WordPress dashboard.
